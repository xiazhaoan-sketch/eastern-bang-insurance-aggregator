## üí° Lessons Learned

* **Coordination in a Large Team:** With five members in a class project, aligning schedules for synchronous work (like task division and prioritization) was a significant challenge. We learned we need to rely more on asynchronous communication tools or establish a single, recurring weekly "sync" time that everyone respects.

* **Leveraging AI for Efficiency:** The strategic use of AI-powered tools (like code assistants and documentation generators) proved highly effective. It significantly accelerated our development velocity and allowed the team to focus on complex logic rather than boilerplate tasks.
